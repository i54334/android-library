# Install

## gradle

```gradle
    compile 'com.github.axet:android-library:1.4.4'
```

## maven

```xml
<dependency>
  <groupId>com.github.axet</groupId>
  <artifactId>android-library</artifactId>
  <version>1.4.4</version>
  <type>aar</type>
</dependency>
```
